William & Mary Studies

WMStudies aims to inform you whenever a class you are following opens up.

It does this via webscrapping. Put simply, it manually goes through all of the classes and checks to see their status, sending you a message via SMS whenever a class you follow changes its status

Sign Up

To start, register your username and phone number, which is needed to send you messages. The only time that your phone number will be used is for informing you of class status changes. Only U.S. phone numbers are compatible at this time.

Classes

Once you have signed up, simply go to Class Search and type in the CRNs of all of the courses that you would like to follow. If you input your phone correctly, you will receive a message when the status changes.
